Security

Shift left, every gate

Six security checks run before any code reaches staging. Zero secrets live in the cluster — IRSA means pods assume IAM roles directly.

01 / SAST
Static Analysis
Semgrep · ESLint · Hadolint
Catches hardcoded secrets, injection flaws, and Dockerfile anti-patterns before the image is ever built.
02 / SCA
Dependency Scan
Snyk · OWASP Dependency-Check
Audits npm/pip transitive deps for known CVEs. Fails on CRITICAL or HIGH severity findings.
03 / Image
Container Scan
Trivy · ECR scan-on-push
Scans OS packages and app layers in the built image. Generates and signs an SBOM for every release tag.
04 / IaC
Infra Scan
tfsec · Checkov
Validates Terraform modules for insecure patterns — public S3 buckets, missing encryption, overly permissive SGs.
05 / DAST
Dynamic Analysis
OWASP ZAP
Active scan against the running staging deployment. Tests for OWASP Top 10 including SQLi, XSS, auth bypass.
06 / Secrets
Zero Stored Creds
AWS IRSA · Secrets Manager
No static keys in Kubernetes Secrets. Pods use IRSA to assume IAM roles and pull from Secrets Manager at runtime.
Observability

See everything, miss nothing

◉
Prometheus
Scrapes metrics from all three microservices, the EKS node exporter, and the kube-state-metrics endpoint. Custom SLO recording rules defined per service.
node-exporter kube-state-metrics service monitors SLO rules
▦
Grafana
Three dashboards: cluster health, application golden signals (latency, traffic, errors, saturation), and deployment tracking. Alerts route to Slack in staging, PagerDuty in production.
golden signals deploy markers Slack alerts PagerDuty
📋
Centralised Logging
Fluent Bit ships pod logs to CloudWatch Logs. Log groups per namespace, retention policy per environment. Structured JSON output from all services.
Fluent Bit CloudWatch structured JSON
🎯
Argo Rollouts Analysis
Canary deployments query Prometheus for error rate and p99 latency. If either breaches threshold during rollout, Argo automatically aborts and rolls back without human intervention.
analysis templates auto rollback p99 latency error rate SLO
